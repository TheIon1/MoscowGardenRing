Task 1:
Представьте систему, состоящую из сервиса авторизации, сервиса профиля пользователя и сервиса уведомлений. Предложите план тестирования взаимодействия этих сервисов при смене пароля пользователем.

Task 2:
Рассмотрите веб-приложение, использующее отдельные сервисы для загрузки изображений и для аналитики поведения пользователей. Сформулируйте набор тестов, проверяющих корректность работы при одновременной загрузке большого количества изображений.
